# Images Directory

## Required Images

Upload the following images to this directory:

### 1. **logo-unmul.png**
- Logo Universitas Mulawarman
- Recommended size: 200x200 pixels (or larger)
- Format: PNG with transparent background
- Location: Left side of navigation bar

### 2. **logo-blu.png**
- Logo BLU (Badan Layanan Umum)
- Recommended size: 200x200 pixels (or larger)
- Format: PNG with transparent background
- Location: Right side of navigation bar

### 3. **favicon.png**
- Favicon untuk browser tab
- Recommended size: 32x32 pixels or 64x64 pixels
- Format: PNG
- Will be displayed in browser tabs and bookmarks

### 4. **og-image.jpg**
- Open Graph image for social media sharing (Facebook, WhatsApp, Twitter, etc.)
- Recommended size: 1200x630 pixels
- Format: JPG or PNG
- This image will appear when someone shares the iLab UNMUL website link

---

## Fallback Behavior

If the images are not uploaded:
- Logo UNMUL will show a blue circular placeholder with "UNMUL" text
- Logo BLU will show an orange circular placeholder with "BLU" text
- Favicon will use browser default
- OG image will not display (social media will use default preview)

---

## How to Replace Images

1. Download or prepare your logo files
2. Rename them exactly as shown above
3. Upload to this `/public/images/` directory
4. Refresh the website to see the changes

**Note:** The navigation bar will automatically detect and use the uploaded images. No code changes needed!
