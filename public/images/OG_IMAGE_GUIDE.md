# OG Image Guide untuk iLab UNMUL

## Spesifikasi OG Image

### Ukuran & Format
- **Ukuran**: 1200 x 630 pixels (rasio 1.91:1)
- **Format**: JPG atau PNG
- **Ukuran File**: Maksimal 8MB (rekomendasi < 500KB untuk loading cepat)
- **DPI**: 72 dpi (untuk web)

## Instalasi

1. Buat gambar sesuai spesifikasi di atas
2. Export sebagai **og-image.jpg**
3. Simpan di folder: `public/images/og-image.jpg`
4. Pastikan ukuran file < 500KB untuk performa optimal

## Testing OG Image

Setelah upload, test OG Image di:

1. **Facebook Debugger**: https://developers.facebook.com/tools/debug/
2. **Twitter Card Validator**: https://cards-dev.twitter.com/validator
3. **LinkedIn Post Inspector**: https://www.linkedin.com/post-inspector/
4. **Open Graph Check**: https://www.opengraph.xyz/
